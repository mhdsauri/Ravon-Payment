# Ravon Payment

Ravon Payment adalah halaman utilitas sederhana yang digunakan untuk mempermudah pengguna **menyalin nomor pembayaran** dan **mengunduh QRIS**.  
Website ini **tidak memproses transaksi**, melainkan hanya menampilkan informasi pembayaran secara ringkas dan mudah diakses.

## Fitur
- Salin otomatis nomor pembayaran (Copy to Clipboard)
- Unduh QRIS pembayaran
- Tampilan responsif (mobile & desktop)
- Toast notification untuk konfirmasi aksi pengguna

## Tujuan
Proyek ini dibuat sebagai **alat bantu pembayaran**, agar proses transfer dapat dilakukan dengan lebih cepat dan mengurangi kesalahan input nomor pembayaran.

## Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome

## Disclaimer
Website ini hanya menyediakan informasi pembayaran dan **tidak memproses, menyimpan, atau memverifikasi transaksi apa pun**.  
Seluruh proses pembayaran dilakukan di luar website ini.
